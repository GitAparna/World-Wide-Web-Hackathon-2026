**Montgomery Circular Tech Dashboard** 🌱

Prototype built during the World Wide Web Hackathon 2026 exploring how cities can use data-driven insights to transform environmental and economic challenges into opportunities.

**Problem**

Cities with growing data center infrastructure face increasing electronic waste, while abandoned land remains underutilized and communities need sustainable job opportunities.

**Solution & Impact**

The Montgomery Circular Tech Dashboard proposes a data-driven portal that helps identify opportunities to:

  -Repurpose abandoned land

  -Support e-waste recovery and refurbishing hubs

  -Enable cooling technology innovation

  -Create local job opportunities through partnerships with tech companies and stakeholders

  -Generate Revenue to promote further economic growth

**How to Run**

Clone the repository:

  git clone https://github.com/GitAparna/World-Wide-Web-Hackathon-2026.git

Navigate to the project folder:

  cd World-Wide-Web-Hackathon-2026

Install dependencies (if applicable):

  npm install

Run the development server:

  npm run dev

Open in your browser:

  eg: http://localhost:8080/  
  
  Hints: Adjust to the right port number if needed.
  Hints: Adjust your environment variable to connect to Bright Data Apis
